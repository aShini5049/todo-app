Todo List app - A simple and responsive todo-app application built with React/Next.js and Tailwind CSS.

How to run the project:
1.	Clone the repository
git clone https://github.com/YOUR_USERNAME/todo-list-app.git
2.	Install Dependencies
npm install
3.	Start the development server
npm run dev
4.	Open your browser and go to http://localhost:3000/

Features:
-	Add new tasks
-	Mark tasks as completed
-	Delete tasks
-	Filter tasks by status: All, Completed, Pending
-	Tasks persist using localStorage, so data is saved even after page reload
-	Responsive layout using Tailwind CSS

Technologies
-	React / Next.js
-	Tailwind CSS
